 Fake Currency Detection System 
This project is a machine learning-based web application that detects whether a currency note is real or fake.

 Features
- Upload currency image
- Detect fake or real note
- Simple web interface using Flask

Technologies Used
- Python
- Flask
- Machine Learning
- HTML/CSS

 Full Project Download
 [Download Complete Project](https://drive.google.com/drive/folders/1j3Quu2zLiwd-ry7X63VnBJe2VeIhF5mU)

 How to Run
1. Download the project from the above link  
2. Extract the ZIP file  
3. Install dependencies:  
   pip install -r requirements.txt  
4. Run the app:  
   python app.py
